# Mi 2025 en Música

Dashboard interactivo de mis escuchas de Spotify durante 2025.

## Ver el dashboard

**[https://chekos.github.io/mi-2025-en-musica/](https://chekos.github.io/mi-2025-en-musica/)**

## Características

- Visualizaciones interactivas con Observable Plot
- Heatmaps facetados (small multiples) por mes
- Bubble charts con doble codificación (tamaño + saturación)
- Diseño inspirado en Tufte, Giorgia Lupi y Alberto Cairo

## Datos

Datos de Spotify Extended Streaming History, procesados y convertidos a Pacific Time.

## Tecnologías

- HTML/CSS/JavaScript vanilla
- [D3.js](https://d3js.org/)
- [Observable Plot](https://observablehq.com/plot/)
- GitHub Pages
